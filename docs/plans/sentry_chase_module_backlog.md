## 目标定义

- 行为：在满足追击条件时主动跟进目标，不满足时回退到现有点位策略。

- 可通过参数开关启用/禁用追击功能。

- 速度输出满足限幅/限加速度，不引入抖动和危险突变。
- 不破坏当前 `/ly/navi/goal` 与 `/ly/control/vel` 既有链路。
